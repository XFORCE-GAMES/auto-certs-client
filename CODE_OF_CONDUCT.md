# Code of Conduct

This project adopts the **Contributor Covenant, version 2.1** as its
Code of Conduct.

Canonical text: <https://www.contributor-covenant.org/version/2/1/code_of_conduct/>

We've adopted it verbatim — that link is the normative document. If
the upstream Contributor Covenant ever publishes a revised version,
we'll evaluate whether to adopt the new revision and update this
pointer.

## Scope

The Code applies to all project spaces, including:

- This GitHub repository (issues, pull requests, discussions, the
  wiki if/when enabled, and any review comments).
- Any other public channel where someone is interacting with this
  project on behalf of the maintainer or the broader contributor
  community.

## Reporting

Reports of conduct issues are handled through the same private
channel as security issues — see [SECURITY.md](SECURITY.md).

> Repository home → **Security** tab → **Report a vulnerability**

That channel is intentionally chosen for its privacy properties: it
creates a private thread between the reporter and the maintainer, and
the original report stays confidential.

If the report concerns the maintainer themselves, please email
`doggieliu@xforce-games.com` directly — the GitHub security-advisory
channel goes to the same inbox but a direct email makes the
conflict-of-interest explicit and lets you add a third-party CC if
helpful.

## Enforcement

The maintainer follows the four-tier enforcement ladder described in
the Contributor Covenant 2.1 (Correction → Warning → Temporary Ban →
Permanent Ban). The full ladder is at the canonical URL above.

A few project-specific notes:

- Issues that disclose CP-identifying details (real `app_code` values,
  IP addresses, token-shaped strings) get **closed without warning**
  and the original poster receives a one-time educational message
  explaining the redaction expectation. Repeat instances escalate.
- Issues that contain harassment toward another contributor jump
  straight to the Warning tier (Contributor Covenant ladder rung 2)
  rather than starting at Correction (rung 1).
